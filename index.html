About the Accessibility Checker:

What is this?

  • A place to check the accessibility of a page, for Q2 employees.
    This software uses multiple APIs to run various accessibility tests to ensure that your page is inclusive and accessible for everyone. It includes a dashboard with common statistics to types of errors, as well as sections for the results that each of the APIs return.

APIs Used

WebAIM WAVE API


The WebAIM API will provide errors and accessibility-related components, sorted into categories. Locations of each instance will also be displayed.


WebAIM WAVE

Pa11y API


This API will check against the WCAG AA Guidelines, Version 2.1, and provide feedback and recommendations for each violation found.


Pa11y

A11y API


This API will run a simple series pass-fail tests and determine if they apply to the URL provided. Help documentation will be provided about each of the tests conducted.


A11y

Puppeteer API


The Puppeteer API will visit the URL entered and take screenshots in both a Mobile and Desktop view, and provide them on the screen in a panel.


Puppeteer
